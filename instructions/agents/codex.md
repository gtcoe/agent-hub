# Codex — Agent Notes

## Instruction Source

This AGENTS.md is generated by `.agent-hub/bin/agent-sync.js`. Do not edit it directly.
To change instructions: edit sources in `.agent-hub/instructions/`, then run `node .agent-hub/bin/agent-sync.js`.

## Skills

Skills are loaded from `.agent-hub/skills/` via `skill_paths` in `~/.codex/config.toml`.
Invoke skills by their trigger phrases (e.g. `/caveman`, `/caveman-commit`).

# Claude Code — Agent Notes

## Instruction Source

This CLAUDE.md is generated by `.agent-hub/bin/agent-sync.js`. Do not edit it directly.
To change instructions: edit sources in `.agent-hub/instructions/`, then run `node .agent-hub/bin/agent-sync.js`.

## Skills

Custom skills are available in `.agent-hub/skills/`. The full skill index is at `.agent-hub/generated/skills/README.md`.
Invoke skills by their trigger phrases or slash commands (e.g. `/caveman`, `/caveman-commit`).

## Memory

This workspace uses Claude Code auto-memory at `~/.claude/projects/*/memory/`.
Check memory at the start of complex multi-session tasks to recover prior context.

## Agent Hub Workflow

When modifying agent setup (MCPs, instructions, skills):
1. Edit the relevant registry file in `.agent-hub/registry/`
2. Run `node .agent-hub/bin/agent-sync.js` to regenerate all adapters
3. Run `node .agent-hub/bin/agent-validate.js` to validate
4. Run `node .agent-hub/bin/agent-install.js --dry-run` to preview live config changes
5. Run `node .agent-hub/bin/agent-install.js --apply --agent <name>` to install
